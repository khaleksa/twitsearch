== TwitSearch

=== Project dependencies:

* Ruby version - 2.3.1

* Rails version - 4.2.1

* Rspec - test suite

=== Description:

TwitSearch is a web interface for twitter search engine. It works by entering a search term and returning the top 100 results. In addition to the results, the application shows the top 10 hashtags and let select country used for finding top hashtags.
